# Smart Calculator
A material design inspired Calculator.

# What is it?
<p>It's a simple but smart calculator built in JavaFX using the library <a href="https://github.com/jfoenixadmin/JFoenix">JFoenix</a> for the material design look</p>

# What i learned?
<ul>
<li>While working on the JavaFX project, I learned the MVC pattern to design the calculator</li>
<li>I also learned how to use external libraries in my project</li>
<li>Learned to come-out with different solution to make the app robust and efficient</li>
</ul>
