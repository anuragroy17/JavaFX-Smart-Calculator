# Smart Calculator
A material design inspired Calculator.

### What is it?
It's a simple but smart calculator built in JavaFX using the library <a href="https://github.com/jfoenixadmin/JFoenix">JFoenix</a> for the material design look

### What i learned?
- While working on the JavaFX project, I learned the MVC pattern to design the calculator</li>
- I also learned how to use external libraries in my project</li>
- Learned to come-out with different solution to make the app robust and efficient</li>
